## 🔍 Overview
**The Payload Forge** is a terminal-inspired, React-based editor designed for security researchers and hardware enthusiasts. It provides a streamlined interface for writing, testing, and managing **DuckyScript** payloads across various operating systems.

Whether you're automating routine tasks or demonstrating security vulnerabilities, The Payload Forge offers a sleek, high-contrast workspace that stays out of your way.

---

## ✨ Features
- **🎯 Multi-OS Support:** Extensive library of predefined scripts for Windows, macOS, Linux, Android, and iOS.
- **📟 Terminal Aesthetic:** High-contrast, green-on-black UI designed for maximum focus and low eye strain.
- **🚀 One-Click Actions:** Instant clipboard copy and `.txt` file generation for rapid deployment to your HID device.
- **📝 Live Editor:** Dynamic DuckyScript editor with syntax-agnostic layout and interactive feedback.
- **💎 Modern React Architecture:** Built on React 19 and Tailwind CSS v4 for a blazing-fast, responsive experience.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/the-payload-forge.git
    cd the-payload-forge
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run in development mode:**
    ```bash
    npm run dev
    ```
    *Access the forge at: `http://localhost:3000`*
